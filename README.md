# Babel-Webpack-Bolierplate

## Supports
* `ES6+`
* `SASS`
* Webpack Live Reloading Dev Server

## Webpack
### Plugins
* `HTMLWebpackPlugin`
* `MiniCssExtractPlugin`

### Loaders
* `babel-loader`
* `css-loader`
* `sass-loader`

## Babel Preset
* `@babel/preset-env`

## Getting Started
```bash
# development
$ npm run dev

# production
$ npm run build
```
